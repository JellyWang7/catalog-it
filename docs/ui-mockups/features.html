Platform Features & Capabilities

A detailed breakdown of how CatalogIt helps you organize and collaborate.

List Creation & Control

Flexible Visibility

Set lists as **Private** (for personal use/inventory) or **Public** (for sharing and community engagement).

Rich List Metadata

Add detailed descriptions, primary categories, and custom tags to each list for easy filtering.

Item Enrichment

Detailed Item Notes

Attach long-form notes, rich descriptions, and custom fields to every cataloged **Item**.

Custom Item Categories

Apply granular categories to individual items for advanced searching and reporting.

Community & Feedback

5-Star User Rating

Public lists enable community members to contribute **Ratings** to items, driving list quality.

Activity Logging

Track all changes, comments, and rating contributions in a central, unchangeable **Activity Log**.
