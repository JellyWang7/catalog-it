# Design Documentation

This folder contains the HTML prototypes and design mockups for CatalogIt.

## Files

- **home.html** - Homepage design
- **features.html** - Features page design
- **public-explore.html** - Public lists exploration page
- **public-list-details.html** - Individual public list details page
- **list-details.html** - List details page design
- **knowledge-base.html** - Knowledge base/help page
- **social-media-mockup.html** - Social media integration mockup

## Purpose

These static HTML files serve as design references for implementing the React frontend components. They define the UI/UX, layout, and styling that should be maintained during development.

## Usage

When building React components:
1. Reference these designs for layout structure
2. Extract color schemes and styling
3. Identify reusable component patterns
4. Ensure responsive design matches mockups

The designs use Tailwind CSS classes which can be directly translated to the React + Tailwind implementation.
